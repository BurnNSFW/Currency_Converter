# Currency Converter 

## Автор
Алиев Рамазан  

## Описание

Desktop-приложение для конвертации валют с графическим интерфейсом, использующее внешнее API для получения актуальных курсов.  
Поддерживает сохранение истории конвертаций, загрузку истории при запуске и обновление курсов в реальном времени.

## Функциональность

- Выбор валют из выпадающего списка
- Ввод суммы для конвертации
- Получение актуального курса через API (CurrencyBeacon)
- Отображение результата конвертации
- Сохранение истории в JSON-файл
- Загрузка истории при запуске приложения
- Просмотр истории конвертаций в таблице
- Очистка истории с подтверждением
- Валидация ввода (только положительные числа)
- Обработка ошибок API (нет интернета, неверный ключ)
- Конфиденциальность: API-ключ хранится в `.env` файле

## Как получить API-ключ

1. Перейдите на сайт [CurrencyBeacon](https://currencybeacon.com/)
2. Нажмите кнопку **"Get started for free"**
3. Зарегистрируйтесь (email + пароль)
4. После регистрации вы получите бесплатный API-ключ
5. Скопируйте ключ 

## Установка и запуск

### Требования
- Python 3.7 или выше
- Библиотеки: `requests`, `python-dotenv`

### Инструкция

# 1. Клонируйте репозиторий
git clone https://github.com/yourusername/currency-converter.git
cd currency-converter

# 2. Установите зависимости
pip install -r requirements.txt

# 3. Настройте API-ключ
#    Скопируйте файл-пример и вставьте свой ключ
cp .env_example .env
#    Откройте .env и замените YOUR_API_KEY_HERE на реальный ключ

# 4. Запустите приложение
python main.py
