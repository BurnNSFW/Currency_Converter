# Currency Converter 

## Автор
Алиев Рамазан

## Описание
Desktop-приложение для конвертации валют с графическим интерфейсом, использующее внешнее API для получения актуальных курсов. Поддерживает сохранение истории конвертаций и обновление курсов.

## Функциональность
- Выбор валют из выпадающего списка
- Ввод суммы для конвертации
- Получение актуального курса через API
- Отображение результата конвертации
- Сохранение истории в JSON-файл
- Просмотр истории конвертаций
- Очистка истории
- Валидация ввода (только положительные числа)

## Как получить API-ключ

1. Перейдите на сайт [CurrencyBeacon](https://currencybeacon.com/)
2. Нажмите кнопку "Get started for free"
3. Зарегистрируйтесь
4. После регистрации вы получите бесплатный API-ключ
